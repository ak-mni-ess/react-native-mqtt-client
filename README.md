# react-native-mqtt-client

MQTT client for React Native

## Installation

```sh
npm install react-native-mqtt-client
```

## Usage

```js
import MqttClient from "react-native-mqtt-client";

// ...

const result = await MqttClient.multiply(3, 7);
```
